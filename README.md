# Учёт семей студентов

Лабораторная работа №2 по курсу "Проектирование программного обеспечения интеллектуальных систем"

## Описание предметной области

Программная система предназначена для учёта информации о студентах и их семьях.

Каждая запись содержит:

| Поле | Тип данных | Описание |
|------|------------|----------|
| ФИО студента | str | Фамилия, имя, отчество студента |
| ФИО отца | str | Фамилия, имя, отчество отца |
| Заработок отца | float | Доход отца в рублях |
| ФИО матери | str | Фамилия, имя, отчество матери |
| Заработок матери | float | Доход матери в рублях |
| Число братьев | int | Количество братьев студента |
| Число сестер | int | Количество сестер студента |

## Функциональные возможности

| № | Возможность | Описание |
|----|-------------|----------|
| 1 | Добавление записей | Через диалоговое окно с проверкой ввода |
| 2 | Просмотр записей | Таблица с постраничной навигацией |
| 3 | Поиск записей | По ФИО студента, ФИО родителя, количеству братьев/сестер, доходу |
| 4 | Удаление записей | По заданным критериям с подтверждением |
| 5 | Сохранение данных | В XML-файл (DOM парсер) |
| 6 | Загрузка данных | Из XML-файла (SAX парсер) |

## Демонстрация работы

### Запуск приложения

```bash
python main.py
```

**Пример 1: Просмотр всех записей**

После запуска открывается главное окно с таблицей. Если данные не загружены, нужно нажать **Загрузить** и выбрать XML-файл из папки `resources/data/`. Таблица отображает записи постранично. Внизу можно выбрать количество записей на странице (5, 10, 20, 50) и переключать страницы кнопками `<<`, `<`, `>`, `>>`.

**Пример 2: Добавление новой записи**

1. Нажать кнопку **Добавить** на панели инструментов (или `Ctrl+A`).
2. Заполнить поля:
   - Студент: Щурко Ксения Витальевна
   - Отец: Щурко Виталий Владимирович, заработок 100000
   - Мать: Щурко Анжелика Викторовна, заработок 80000
   - Братья: 1, Сёстры: 0
3. Нажать кнопку **Добавить**.
4. Запись появляется в таблице.

**Пример 3: Поиск по фамилии студента**

1. Нажать кнопку **Поиск** (или `Ctrl+F`).
2. Поставить галочку **Использовать** в блоке *По ФИО студента*.
3. Ввести *Иванов* в поле значения.
4. Нажать кнопку **Найти**.
5. В правой панели отобразятся все студенты с фамилией Иванов.

**Пример 4: Поиск по диапазону дохода отца**

1. Открыть диалог поиска.
2. Поставить галочку **Использовать** в блоке *По доходу родителя*.
3. Выбрать *father* (отец).
4. Ввести *Доход от: 50000*, *Доход до: 100000*.
5. Нажать кнопку **Найти**.
6. Отобразятся студенты с доходом отца от 50 000 до 100 000 рублей.

**Пример 5: Удаление по количеству братьев**

1. Нажать кнопку **Удалить** (или `Ctrl+D`).
2. Поставить галочку **Использовать** в блоке *По числу братьев/сестер*.
3. Выбрать *brothers* (братья), ввести *0*.
4. Нажать кнопку **Удалить**.
5. Подтвердить действие в появившемся окне.
6. Получить сообщение *Удалено записей: N*.
7. В таблице остались только студенты, у которых есть братья.

**Пример 6: Сохранение и загрузка**

1. Добавить несколько записей или загрузить существующий файл.
2. Нажать кнопку **Сохранить** (или `Ctrl+S`), указать имя файла, например *test.xml*.
3. Появится сообщение *Сохранено N записей*.
4. Нажать кнопку **Загрузить** (или `Ctrl+O`), выбрать другой XML-файл.
5. Таблица обновится новыми данными.
